О проекте

Этот проект помогает вести личный учет операций с ценными бумагами.

В нем можно хранить покупки, продажи, комиссии и планируемые цены.

Главная страница показывает активные позиции и текущий результат.

Архив собирает завершенные операции, которые больше не нужны на главной.

Для каждой бумаги сохраняются тикер, название, тип и дополнительные параметры.

Брокеры вынесены отдельно, чтобы учитывать разные комиссии.

Расчеты помогают быстро увидеть прибыль, убыток и желаемый результат.

Проект поддерживает авторизацию, поэтому данные привязаны к пользователю.

Административная панель используется для управления справочниками и сделками.

Также есть раздел загрузки файлов для хранения связанных документов.

Интеграция с T-Invest используется для получения актуальных рыночных цен.

Интерфейс построен на Django и Bootstrap, поэтому его легко развивать дальше.

Структура проекта разделяет настройки, маршруты, шаблоны, формы и модели.

Модели вынесены в отдельные файлы, чтобы доменная логика оставалась понятной.

SQLite используется как простая база данных для текущего небольшого объема данных.

Gunicorn обслуживает приложение на сервере через системный сервис.

Проект можно постепенно расширять отчетами, фильтрами и импортом операций.

Главная цель приложения — быстро понимать состояние портфеля без лишних таблиц.

Страница “О нас” описывает назначение системы и помогает новым пользователям сориентироваться.

Дальнейшее развитие можно строить вокруг безопасности, удобства и автоматизации расчетов.
